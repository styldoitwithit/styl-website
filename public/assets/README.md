# Client Logo Assets

Place your client logo images in this directory.

Filenames must match exactly (case-sensitive):

- `ATLAS HOSPITALS.png`
- `DR ANILKUMAR EYE HOSPITAL.jpg`
- `MUKESH ARTHRO CARE HOSPITAL.png`
- `SRI RAMAKRISHNA SPECIALTY HOSPITALS.jpg`

Until images are added, the site will display placeholder text boxes with the client name.

## Recommended Specs
- Format: PNG (preferred) or JPG
- Dimensions: ~300×150px minimum
- Background: Transparent (PNG) or white/dark
